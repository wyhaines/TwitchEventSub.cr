# twitch_event_sub

This implements a simple library to wrap up interaction with the Twitch EventSub API.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     twitch_event_sub:
       github: your-github-user/twitch_event_sub
   ```

2. Run `shards install`
